# 🛰️ CrimeReport

**CrimeReport** is an open-source civic tech platform that lets users report and view crimes on a map using geolocation. Each report includes the nature of the crime, time, and description. All users can view red flags marking the crime zones.

## 📌 Features
- Report crimes with map marker and details
- See crime flags on a map
- Anonymous reporting
- Aggregated and real-time updates

## 🧰 Tech Stack
- **Frontend**: React.js / Next.js + Leaflet or Google Maps
- **Backend**: NestJS / Node.js or PHP
- **Database**: PostgreSQL with PostGIS or MongoDB
- **Deployment**: Vercel, Railway, Supabase
